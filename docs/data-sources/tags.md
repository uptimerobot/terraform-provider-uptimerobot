# uptimerobot_tags (Data Source)

Lists existing UptimeRobot monitor tags with an optional exact `name` filter.

Use this data source to discover tag IDs for read-only composition, reporting, or future API features that reference tags by ID. Tags are still owned through monitor configuration via `uptimerobot_monitor.tags`; deleting tags is out of scope for this provider surface.

## Example Usage

```terraform
data "uptimerobot_tags" "all" {}

data "uptimerobot_tags" "production" {
  name = "production"
}

output "all_tag_ids" {
  value = data.uptimerobot_tags.all.ids
}

output "production_tag_ids" {
  value = data.uptimerobot_tags.production.ids
}
```
