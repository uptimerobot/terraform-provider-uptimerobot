# uptimerobot_psp (Data Source)

Looks up one existing UptimeRobot Public Status Page (PSP) by `id` or exact `name` without managing it. PSP names are not guaranteed unique in UptimeRobot; if more than one PSP has the configured name, the data source returns an error and lists the matching IDs. Configure `id` to select one PSP unambiguously.

Use this when a status page is managed in another Terraform state, created through the UI/API, or otherwise owned outside the current configuration. The data source exposes read-only PSP metadata and settings so existing status pages can be referenced from related resources such as `uptimerobot_psp_announcement`.

The UptimeRobot API does not return PSP passwords. Use `is_password_set` to determine whether password protection is enabled.

## Example Usage

```terraform
data "uptimerobot_psp" "production" {
  name = "Example.com Status"
}

resource "uptimerobot_psp_announcement" "maintenance" {
  psp_id     = tonumber(data.uptimerobot_psp.production.id)
  title      = "Scheduled maintenance"
  content    = "Maintenance is scheduled for tonight."
  status     = "pending"
  type       = "maintenance"
  start_date = "2030-01-01T00:00:00Z"
}

output "status_page_url_key" {
  value = data.uptimerobot_psp.production.url_key
}
```

## Schema

### Optional

- `id` (String) The PSP ID. Configure this for an exact lookup, or omit it and configure `name`.
- `name` (String) The exact PSP name. PSP names are not guaranteed unique; if multiple PSPs match, configure `id` instead.

### Read-Only

- `auto_add_monitors` (Boolean) Whether the PSP automatically includes all current and future monitors.
- `custom_domain` (String) Custom domain configured for the PSP, or null when no custom domain is set.
- `custom_settings` (Attributes) Custom settings for the PSP. (see [below for nested schema](#nestedatt--custom_settings))

- `is_password_set` (Boolean) Whether a password is set for the PSP. The password value itself is not returned by the UptimeRobot API.
- `monitor_ids` (Set of Number) Monitor IDs assigned to the PSP.

- `tag_ids` (Set of Number) Monitor tag IDs assigned to the PSP.
- `url_key` (String) URL key for the PSP.
- `use_small_cookie_consent_modal` (Boolean) Whether the small cookie consent modal is enabled.
